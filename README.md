# central-28circle-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面彼此独立，不针对任何特定网站或域名，仅作为静态资源存储与展示。

## 项目简介

`central-28circle-lab` 是一个集中存放 HTML 页面的仓库。所有页面均为纯前端实现，不依赖后端服务。仓库本身不提供任何特定功能或服务，仅作为页面存档与分发的载体。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库首页（可选，用于导航）
└── pages/             # 存放各独立 HTML 页面的目录
    ├── page1.html
    ├── page2.html
    └── ...
```

- `index.html`：可选，用于提供页面列表或导航。
- `pages/`：所有独立页面的存放位置。每个文件为一个完整的 HTML 页面，可单独访问。

## 页面归档说明

- 所有页面均为静态 HTML，无需服务器环境即可在浏览器中打开。
- 页面之间无依赖关系，可独立使用或引用。
- 页面内容可能随时间更新或移除，请以仓库最新版本为准。

## 维护说明

- 本仓库由维护者不定期更新。
- 欢迎提交 Issue 或 Pull Request 来报告问题或建议改进。
- 新增页面请放入 `pages/` 目录，并确保文件名唯一且有意义。
- 不建议在页面中嵌入外部资源（如 CDN 脚本），以免影响离线访问。
